# MeleeCombat

First person melee combat prototype.
Made in Unreal Engine 5

<p align="center">
  <img src="assets/gifs/gameplay.gif"><br/>
</p>

## Features

- **Player**
  1. Attacking (LMB)
  2. Blocking (RMB)  
- **Enemies** Enemies choose one of the two options randomly
  1. Attacking
  2. Blocking
